Portfolio - Node Express Nodemailer
A modern, responsive emailsender.


Getting Started
Prerequisites
Node.js (v18+)
npm or yarn
Installation
Navigate to the project directory:
cd portfolio
Install dependencies:
npm install
Create a .env file (copy from .env.example):
VITE_BACKEND_URL=http://localhost:8000
Development
Run the development server:

npm run dev
